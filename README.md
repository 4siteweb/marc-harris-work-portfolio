# Marc Harris — Work Portfolio

A static portfolio website designed for GitHub Pages and for linking from a CV.

## What is included

- Professional landing page
- Capability summary
- Featured Flickr evidence albums
- Full archive of 39 supplied Flickr album links
- Responsive mobile layout
- Simple category filters
- Placeholder CV download link

## Add your CV

Place your latest PDF at:

`assets/Marc-Harris-CV.pdf`

A small placeholder file is currently included only to preserve the folder structure.

## Edit projects

Open `projects.js`.

- `featuredProjects` controls the polished project cards.
- `flickrArchive` contains every historical album link.

Copy an existing project object, change the title, description, category and URL, then commit the update.

## Publish on GitHub Pages

1. Create a public GitHub repository, for example `marc-harris-portfolio`.
2. Upload all files from this folder.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. Add the generated GitHub Pages URL to your CV.

Suggested CV wording:

**Project Portfolio:** `https://YOUR-USERNAME.github.io/marc-harris-portfolio/`

## Important

Only include project photographs that you are permitted to publish. Remove any photographs showing confidential information, personal data, security systems, access codes, drawings or client-restricted areas.
